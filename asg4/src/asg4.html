Notes for Graders

Only the sphere has specular lighting, as per the professor's comment at the end of video "4.6 Add Lighting".

There is also a spot light. Both the point and spot lights can be turned on and off.

My blocky animal, the black cat, is included in the scene.

You can turn off the point light's animation to manually change its X value (Its Y and Z values can be changed with the sliders regardless).

Please use a browser that supports "canvas"
